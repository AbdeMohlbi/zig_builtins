## Zig Builtins
this repo serves as a reference for how to use [builtin-Functions](https://ziglang.org/documentation/master/#Builtin-Functions) in zig, inspired by [ziglings](https://codeberg.org/ziglings/exercises).

at the moment those are the builtin that this repo contains / have tests for

- [x] @addWithOverflow
- [x] @subWithOverflow
- [x] @mulWithOverflow
- [x] @as
- [x] @select
- [x] @sizeOf
- [x] @truncate
